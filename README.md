# NSE4 Libraries
This repository holds libraries used by [NSE4 Core](https://github.com/Alessandro-Salerno/NSE4-Core) and other related projects on the UMSR Minecraft Server.

## How to install
For the latest Python distributions, use this command:
```
pip install git+https://github.com/Alessandro-Salerno/NSE4-Libraries.git#subdirectory=python/mcom && \
    pip install git+https://github.com/Alessandro-Salerno/NSE4-Libraries.git#subdirectory=python/unet
```
